終端機:
安裝 git
直接輸入 git 之後，他會自己開始跑

安裝 brew, 安裝之前要裝 git, brew 在安裝的時候會需要
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安裝 iterm2: 可以透過 brew 但好像也可以不用? 通常都是直接從官網下載了
profile 的同步方式: 可以直接透過 Preferences -> General -> Preferences 裡面的 Load preferences from a custom folder or URL 直接和備份的 profile 同步即可
com.googlecode.iterm2.plist 要複製過去的地方是 
cp com.googlecode.iterm2.plist ~/Library/Preferences/com.googlecode.iterm2.plist

安裝 oh-my-zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

NEXT 目前設定階段停在這裡，安裝/同步完上述的項目後 iterm2 應該已經可以用 option 按鍵左右移動 helloWorld 等的字串單位了

接著是rc那些:
1. 複製 .zshrc-for-mac 到 ~/.zshrc
2. 複製 .zshrc.pre-oh-my-zsh-for-mac 到 ~/.zshrc.pre-oh-my-zsh
3. 複製 .bashrc-for-mac 到 ~/.bashrc
4. 複製 .bash-git-for-mac-for-mac 到 ~/.bash-git
5. 複製 .git-completion.bash 到 ~/.git-completion.bash
6. 複製 .gitconfig-for-mac 到 ~/.gitconfig
7. 創建新的/複製舊的 .bash-work-for-mac 到 ~/.bash-work
8. 創建 .vimrc -> 內容很少，所以還好 ( nu, autoindent, tabstop=2, hlsearch)
9. 把 s 綁到環境變數裡面 ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/.
10. 安裝字體: brew tap homebrew/cask-fonts && brew install --cask font-sauce-code-pro-nerd-font
11. 調整 preference -> profiles -> text -> font -> 選擇 SauceCodePro Nerd Font 開頭的應該都可以
12. 重啟 iterm

git 的權限部分，原本以為只要用 ssh-keygen 就可以了
沒想到還要設定一個 known_host 的項目
```bash
ssh-keygen -t rsa -C "yayayahahahaooii@gmail.com"
ssh-keyscan -t rsa github.com >> ~/.ssh/known_hosts
cat ~/.ssh/id_rsa.pub | pbcopy
```
然後到 github 上面設定就可以了
source: https://stackoverflow.com/questions/13363553/git-error-host-key-verification-failed-when-connecting-to-remote-repository
